## WebRTC 134.5 (2025/10/30)

WebRTC library for Android starting at SDK 21.  This library also provides SQLcipher and Java support for libwebsockets.
Sources used to build this library are available in:

- https://github.com/Twinlife/webrtc
- https://github.com/Twinlife/libwebsockets
- https://github.com/Twinlife/sqlcipher
